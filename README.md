# 🏛️ Obsidian Libraries

> A curated collection of Obsidian knowledge vaults, ready to clone and open in your Obsidian app.

This repository contains ready-to-use Obsidian vaults on various topics. Each vault is a fully self-contained Obsidian workspace with its own notes, configuration, themes, and plugins — so you can start learning and exploring immediately.

## 🚀 How to Use

### Option 1: Clone the whole library (recommended)

```bash
git clone https://github.com/Winters-Glade/Obsidian-Libraries.git
```

Then in Obsidian:
1. Click **Open another vault** (the folder icon in the bottom-left)
2. Select **Open folder as vault**
3. Navigate to the cloned repository and select a vault folder (e.g., `Obsidian-Libraries/History102`)
4. Click **Open**

### Option 2: Download a single vault

If you only want one vault without cloning the entire repo, you can use a tool like [Download Directory](https://download-directory.github.io/) by pasting the vault folder URL.

**Example:** `https://github.com/Winters-Glade/Obsidian-Libraries/tree/main/History102`

## 🤝 Contributing

Have an Obsidian vault you'd like to share? Feel free to:
1. Fork this repository
2. Add your vault as a new folder at the root level
3. Open a Pull Request

Please ensure your vault:
- Has a `README.md` explaining what it contains
- Includes its `.obsidian` configuration for a ready-to-use experience
- Contains no sensitive or personal information
