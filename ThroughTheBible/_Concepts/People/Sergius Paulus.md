# Sergius Paulus

## Description

Sergius Paulus was the Roman proconsul of Cyprus who converted to Christianity after witnessing Paul's confrontation with Elymas the sorcerer. Chuck Smith notes this conversion as a key event in Paul's first missionary journey.
