# Philippi

## Description

Philippi was a Roman colony in Macedonia where Paul founded the first church in Europe. Key events include the conversion of Lydia, the healing of a demon-possessed slave girl, and the imprisonment of Paul and Silas, which led to the conversion of the Philippian jailer (Acts 16).
